    课件最佳学习环境：

　　　　1.软件要求

　　　　需要 IE 5.5 以上版本浏览器，Flash7.0插件。

　　　　2.硬件配置要求：

　　　　CPU 主频300MHz或以上Intel及兼容CPU
　　　　分辨率 800×600以上
　　　　内存 64MB
　　　　显存 4MB
　　　　光驱 8倍速以上
　　　　声卡 SB Pro 兼容声卡
　　　　操作系统： Windows 9x/Me/NT/2000/XP
